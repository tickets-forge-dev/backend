As your ticket count grows, folders and tags help you stay organized.

## Folders

Folders group related tickets into collapsible sections in the feed — like project folders on your desktop.

### Creating a Folder

Click **"+ Folder"** at the top of the ticket feed, type a name, and press Enter.

### Moving Tickets

| Method | How |
|--------|-----|
| **Drag and drop** | Drag any ticket onto a folder header |
| **Right-click menu** | Right-click a ticket → "Move to..." → pick a folder |

To move a ticket back out, drag it to **"Unfiled"** or use the right-click menu.

### Folder Visibility

| Scope | Who sees it |
|-------|-------------|
| **Team** | Everyone on the team |
| **Private** | Only you |

### Managing Folders

Right-click a folder to:
- **Rename** it
- **Reorder** — drag folders up or down
- **Delete** — tickets inside move back to the main feed

Your expand/collapse preferences are personal — collapsing a folder for yourself doesn't affect anyone else.

## Tags

Tags are colored labels you can attach to any ticket for quick categorization.

### Adding Tags

Click the **tag icon** on a ticket row or in the ticket detail view. Type a name, pick a color, and press Enter.

### Filtering by Tags

Select one or more tags in the dashboard filters to narrow your view. Tags appear as colored badges on each ticket row.

### Team-Wide Tags

Tags are shared across the team — everyone sees the same set of tags and can create new ones.
